# EW2026-Afericao

Repositorio base para a ficha de afericao de EngWeb 2026.

## Estrutura

- `dataset_reparacoes.json`: dataset fornecido no enunciado
- `engweb2026-afericao.pdf`: enunciado
- `queries.txt`: queries MongoDB do exercicio 1.2
- `PR.txt`: notas de entrega e instrucoes de execucao
- `ex1/`: API de dados
- `ex2/`: interface
